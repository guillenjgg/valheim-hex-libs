# Contributing to valheim-hex-libs

Thank you for your interest in contributing! Here are some guidelines to help you get started.

## Code Standards

- **Language Version:** .NET Framework 4.8
- **Style:** Follow the `.editorconfig` file conventions
- **Naming:** Use PascalCase for public members, camelCase for private members
- **Comments:** Write clear, concise comments for complex logic

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/guillenjgg/valheim-hex-libs.git
   cd valheim-hex-libs
   ```

2. Open the solution in Visual Studio or your preferred IDE

3. Build the project:
   ```bash
   dotnet build
   ```

## Development Process

1. Create a feature branch: `git checkout -b feature/your-feature-name`
2. Make your changes
3. Commit with descriptive messages: `git commit -m "Add feature description"`
4. Push to your fork and create a Pull Request

## Pull Request Process

- Describe the changes clearly in your PR description
- Link any related issues
- Ensure the code builds without warnings
- Follow the existing code style and conventions

## Issues

Found a bug? Have a feature request? Please open an issue with:
- A clear description
- Steps to reproduce (for bugs)
- Expected vs. actual behavior
